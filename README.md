# Dog Vs Cat Classification using CNN

This project implements a Convolutional Neural Network (CNN) to classify images of dogs and cats. The model is built using TensorFlow/Keras and trained on an image dataset containing dog and cat images.

## Features
- Image classification using CNN
- Data preprocessing and normalization
- Model training and validation
- Accuracy and loss visualization
- Prediction on new images

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Google Colab

## Dataset
The dataset consists of labeled images of dogs and cats used for training and testing the CNN model.

## Model Architecture
- Convolutional Layers
- Batch Normalization
- Max Pooling Layers
- Dense Layers
- Dropout Layer
- Softmax/Sigmoid Output Layer

## Results
The model achieves good accuracy in distinguishing between dog and cat images.

## How to Run
1. Clone the repository.
2. Install the required libraries.
3. Open the notebook in Google Colab or Jupyter Notebook.
4. Run all cells to train and evaluate the model.
